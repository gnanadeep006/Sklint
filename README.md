# SklinT Website Project

This repository contains the SklinT Django website project, including templates, static assets, and app code.

## Project Structure
- `project/`: Django project configuration, routes, and site views.
- `clints/`: App models, admin setup, forms, and app-specific URLs/views.
- `templates/`: HTML templates for the website pages.
- `statics/`: CSS, JavaScript, fonts, and image assets.
- `media/`: Uploaded media used by the site.

## Local Setup
1. Create and activate a Python virtual environment.
1. Install the project dependencies.
1. Run migrations.
1. Start the Django development server.

## Notes
- Keep secrets and environment-specific settings out of version control.
- `db.sqlite3` is the local development database.
